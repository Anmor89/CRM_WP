# West Part CRM

CRM для отдела продаж West Part — прототип.

Одностраничное веб-приложение (один файл `index.html`). Открывается в любом
современном браузере, данные хранятся локально в браузере пользователя.

## Разделы

- **Дашборд** — ключевые показатели, воронка, распределение по менеджерам
- **Воронка** — сделки по этапам (kanban), лента активности
- **Клиенты** — база клиентов менеджеров
- **Полевые продажи** — база торговых представителей, визиты, свод
- **Продажи** — выручка, план/факт
- **Календарь** — запланированные события
- **Договоры** — договорная работа
- **Аналитика** — графики и метрики
- **Выгрузки** — экспорт/импорт данных
- **Настройки / Доступы** — администрирование (только для админа)

## Запуск

Открыть `index.html` двойным кликом в браузере.
